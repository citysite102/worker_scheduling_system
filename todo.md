# 移工排班與工時計算系統 - 功能清單

## 資料庫與後端

- [x] 設計資料庫 Schema（Workers, Clients, Availability, Demands, Assignments）
- [x] 建立測試資料（至少 8 位移工含 2 位停用、3 位客戶、8 筆需求單、10 筆排班）
- [x] 實作移工管理 API（新增、編輯、停用、列表、搜尋、狀態篩選）
- [x] 實作客戶管理 API（新增、編輯、停用、列表、搜尋）
- [x] 實作可排班時間管理 API（每週滾動更新、多時段支援、確認狀態）
- [x] 實作用工需求管理 API（建立、編輯、列表、人力可行性計算）
- [x] 實作需求單指派 API（可指派清單、不可指派原因、排序邏輯、送出前檢查）
- [x] 實作時段重疊檢查邏輯（排班衝突判斷、實際工時衝突標記）
- [x] 實作實際工時回填 API（填寫實際時間、自動計算差異、狀態更新）
- [x] 實作報表輸出 API（移工薪資報表 CSV、客戶工時報表 CSV、日期區間篩選）

## 前端介面

- [x] Dashboard 儀表板（今日需求列表、今日已指派清單、缺口與人數統計）
- [x] 移工管理頁面（列表、新增/編輯表單、搜尋與篩選、詳情頁）
- [x] 客戶管理頁面（列表、新增/編輯表單、搜尋、詳情頁）
- [ ] 可排班時間管理頁面（週選擇器、移工時段編輯、多時段支援、確認狀態顯示）
- [ ] 用工需求管理頁面（列表、新增/編輯表單、人力可行性顯示）
- [x] 需求單指派頁 UX 強化（三區清單、排序邏輯、灰底禁選、衝突原因顯示）
- [x] 智能指派操作（一鍵湊滿、清空已選、已選統計、送出前檢查）
- [ ] 實際工時回填頁面（依日期或需求查看、填寫實際時間、自動計算差異）
- [ ] 報表輸出頁面（移工薪資報表、客戶工時報表、CSV 下載、日期篩選）

## 全站語系與格式

- [ ] 全站介面文字使用繁體中文（台灣）
- [ ] 日期格式統一為 YYYY/MM/DD
- [ ] 時間格式統一為 24 小時制 HH:mm
- [ ] 名詞對照表應用（Worker=移工、Client=客戶、Demand=需求單等）
